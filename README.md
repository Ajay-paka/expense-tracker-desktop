# Expense Tracker Desktop Application

A professional desktop expense tracking application built using Python and Tkinter.

## Features
- User Login System
- Interactive Dashboard
- Expense Analytics Cards
- Smart Search & Filtering
- Column Sorting
- Edit / Delete with Autofill
- Pie Chart Visualization
- Excel Export
- Clean Sidebar UI
- Packaged as Executable (.exe)

## Tech Stack
- Python
- Tkinter
- SQLite
- Pandas
- Matplotlib
- Pillow

## How to Run
pip install -r requirements.txt
python login.py
