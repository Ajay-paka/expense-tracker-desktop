# Expense Tracker Desktop Application 💻

Expense Tracker Desktop is a professional desktop-based personal finance application built using Python and Tkinter.
It helps users securely manage expenses, analyze spending patterns, and generate visual financial insights through an interactive desktop interface.

## 📸 Application Preview
![Expense Tracker(ET)](ETmain.png)

## 🚀 Features

* Secure login system
* Add, edit, and delete expense records
* Smart search and filtering
* Column sorting for better tracking
* Expense analytics dashboard
* Pie chart visualization
* Excel export support
* Clean desktop UI with sidebar navigation
* SQLite-based local data storage

## 🛠 Tech Stack

* Python
* Tkinter
* SQLite
* Pandas
* Matplotlib
* Pillow

## ⚙️ Local Setup

Clone repository:

```bash id="8j5m1x"
git clone https://github.com/Ajay-paka/expense-tracker-desktop.git
cd expense-tracker-desktop
```

Install dependencies:

```bash id="2d7v4q"
pip install -r requirements.txt
```

Run application:

```bash id="1n8r6k"
python login.py
```

## 🧠 Application Flow

1. User logs in
2. Expense dashboard opens
3. User adds income / expense entries
4. Records stored in SQLite database
5. Charts update automatically

## 📊 Core Functionalities

* Expense tracking by category
* Date-based filtering
* Expense summary cards
* Pie chart spending visualization
* Editable records with autofill

## 📦 Export Support

Users can export expense data to Excel for reporting and offline analysis.
